Fencrypt

Secure Your Discord.

Advanced string and media encryption algorithm directly integrated into your server. Keep your sensitive data locked down with UI-based whitelisting.

Command Reference

/encrypt

Encrypt text and optional media. Generates a cipher or a downloadable hash file.

message Required. Target string.
whitelist_ids Optional. Comma-separated Discord IDs.
image_upload Optional. Upload file (max 10MB).
image_link Optional. Image URL (max 10MB).

/decrypt

Decrypt a String or File. Validates against the whitelist automatically.

hash_str Optional. Ciphertext to process.
file Optional. Upload .txt file if hash is too large.
*Must provide either hash_str or file.

Context Menu

Quickly select media from chat without downloading it.

How to use:

  1. Right click any message with an image.
  2. Go to Apps.
  3. Select Choose image.
  4. Run /encrypt to process it.
