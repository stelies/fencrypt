Command Reference

Everything you need to secure your server's communications.

/encrypt

Encrypt text and optional media. Generates a cipher or a downloadable hash file.

message Required Target string to encrypt.
whitelist_ids Optional Comma-separated Discord IDs.
media_upload Optional Upload Image/GIF (max 10MB).
media_link Optional Image/GIF URL (max 10MB).
ephemeral Optional Hide output from others.

/decrypt

Decrypt a String or File. Validates against the whitelist automatically.

hash_str Required* Ciphertext to process.
file Required* Upload .txt file if hash is too large.
ephemeral Optional Hide output from others.
*Must provide either hash_str or file.

Context Menu

Quickly select media (Images/GIFs) from chat without downloading it.

How to use:

  1. Right click any message with media.
  2. Go to Apps.
  3. Select Choose media.
  4. Run /encrypt to process.
